# User Manual

Welcome to Roar! This user manual explains the core functionalities of the platform and how to interact with the system once it is up and running.

## 1. System Overview

Roar is a social media MVP tailored for college sports fans, featuring dynamic video feeds, community interactions, and user profile management. It utilizes a custom Express API backend and a responsive iOS application.

## 2. Core Workflows

### Authentication
1. **Registration**: Upon opening the application for the first time, users can enter a username, email, and password to create an account.
2. **Login**: Returning users can log in securely using their credentials.

### Discovering Content
- **Main Feed**: Upon login, users are presented with a scrollable feed of the latest video content from other users. 
- **Video Interactions**: Tap the screen to play or pause the video. Double-tap to interact with the content directly.

### Engaging with the Community
- **Likes**: Users can 'heart' or like videos they enjoy. Liked videos update dynamically on the backend.
- **Comments**: Tap the comment icon on any video to open the discussion sheet. You can post a new comment or reply to existing ones within the community thread.
- **Follows**: Navigate to another user's profile to follow them. Your feed algorithm prioritizes content from users you follow.

### Uploading Content
- **Video Upload**: Users can hit the '+' action on the navbar to capture or upload a new video. 
- **Metadata**: During upload, provide a title to categorize and contextualize your content. 

### Managing Profile
- **Profile Page**: Navigate to the distinct Profile tab visually reviewing all videos you have personally uploaded.
- **Settings**: Perform basic account management tasks.
